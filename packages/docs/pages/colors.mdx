# Colors

You can using different colors by default or could change it with [`createTheme`](http://localhost:3000/platzily-ui/styling). Many of components get the value of one color. Read well the documentation of the components!

### Examples

You could compose using the props approach like this

```jsx
<Box color="primary"/>
<Box color="text-secondary"/>
<Box color="warning"/>
<Box color="info"/>
<Box color="error"/>
<Box color="text-disabled"/>
```

The code above will result in:

<DefaultStyledBox color="#C4C4C4"/>
</div>
