# Bottons

Buttons allow users to take actions, and make choices, with a single tap.

## Basic Buttons

The `Button` has three variants: `Contained`, `Outlined`, `Text` and is by default: `Contained`.

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto auto', justifyContent: 'center' }}>
  <Button variant="contained">Cotained</Button>
  <Button variant="outlined">Outlined</Button>
  <Button variant="text">Text</Button>
</div>

### Contained Button

`Contained Button` are high-emphasis, is used to actions primary of you app.

```jsx
import { Button } from '@platzily-ui/components';
import { Fragment } from 'react';

const MyComponent = () => {
  return (
    <Fragment>
      <Button>Default</Button>
      <Button variant="contained">Normal</Button>
      <Button variant="contained" disabled>Disabled</Button>
    </Fragment>
  );
};
```

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto auto', justifyContent: 'center' }}>
<Button>Default</Button>
  <Button variant="contained">Normal</Button>
  <Button variant="contained" disabled>Disabled</Button>
</div>

### Outlined Button

## Handling

All components accept handlers that is applied to the root DOM element.

```jsx
import { Button } from '@platzily-ui/components';
import { Fragment } from 'react';

const MyComponent = () => {
  const handleClick = () => {
    alert('clicked');
  };

  return (
    <Fragment>
      <Button onClick={handleClick}>Click me</Button>
    </Fragment>
  );
};
```
