## Use Icons

The `Icons` it add importing the icon and doing the declaration in the you component. Example:

```jsx
import { User } from '@platzily-ui/icons'; //import Icon
import { Frament } from 'react';

export default function MyComponent() {
  return (
    <Frament>
      <User /> //Declaration Icon
    </Frament>
  );
}
```

## Sizes

The `Icons` have prop `size` should change size the icon. Is by default: `1x`

```jsx
import { Link } from '@platzily-ui/icons';
import { Frament } from 'react';

export default function MyComponent() {
  return (
    <Frament>
      <Link size="xs" /> //Size in rem: 0.5rem
      <Link size="sm" /> //Size in rem: 0.75rem
      <Link size="lg" /> //Size in rem: 1rem
      <Link size="1x" /> //Size in rem: 1.5rem
      <Link size="2x" /> //Size in rem: 2.5rem
    </Frament>
  );
}
```

  <Link size="2x" />
</div>

## Sizes Personalized

The `Icons` have props `Height` and `Width` that let change the size to taste.
```jsx
export default function MyComponent() {
  return (
    <Frament>
      <Link height='6rem' Width='6rem' /> //Size in rem: 0.5rem
    </Frament>
  );
}
```

<div style={{display: 'flex', justifyContent: 'center', alignItems: 'center'}}>
  <Airplane height='6rem'/>
</div>
