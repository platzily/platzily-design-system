# Papers

Paper is a basic component used to define some properties to have consistency in container elements.

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto', justifyContent: 'center' }}>
  <PaperComponent elevation={1} />
  <PaperComponent color="secondary" />
  <PaperComponent elevation={4} color="neutral" />
  <PaperComponent variant="outlined" />
</div>

### Basic Paper

```jsx
import { Paper } from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <Paper color='neutral-secondary' />
    <Paper color='neutral' />
    <Paper color='error' />
    <Paper color='info' />
  )
}

```

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto', justifyContent: 'center' }}>
  <PaperComponent color="neutral-secondary" />
  <PaperComponent color="neutral" />
  <PaperComponent color="error" />
  <PaperComponent color="info" />
</div>

### Variant

You could have an outlined paper.

```jsx
import { Paper } from '@platzily-ui/components';

const MyPaper = () => {
  return <Paper variant="outlined" />;
};
```

<div align="center">
  <PaperComponent variant="outlined" />
</div>

### Elevation

The elevation can be used to establish a hierachy between other content. In practical terms, the elevation controls the size of the shadow applied to the surface.

```jsx
import { Paper } from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <Paper elevation={0}>0</Paper>
    <Paper elevation={1}>1</Paper>
    <Paper elevation={2}>2</Paper>
    <Paper elevation={3}>3</Paper>
    <Paper elevation={4}>4</Paper>
    <Paper elevation={5}>5</Paper>
    <Paper elevation={6}>6</Paper>
    <Paper elevation={7}>7</Paper>
  )
}

```

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto', justifyContent: 'center' }}>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={0}>0</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={1}>1</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={2}>2</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={3}>3</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={4}>4</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={5}>5</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={6}>6</PaperComponent>
  <PaperComponent style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }} elevation={7}>7</PaperComponent>
</div>

### Size

With Size prop, you can change the size of Paper to taste. You could compose using the props approach like this:

```jsx
import { Paper } from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <Paper color='neutral-secondary'/>
    <Paper color='primary'/>
  )
}

```

The code above will result in:

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto', justifyContent: 'center' }}>
  <PaperComponent color="neutral-secondary" />
  <PaperComponent color="primary" />
</div>

## Color

The `Paper` change of the color with `color` prop and use values as hexcolor, [colors default](/platzily-ui/colors), HTML color groups, is by default: 'neutral-tertiary'

```jsx
import { Paper } from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <Paper color='secondary'/>
    <Paper color='pink'/>
    <Paper color='#655EBA'/>
    <Paper color='neutral-dark'/>
  )
}

```

<div style={{ display: 'grid', gridTemplateColumns: 'auto auto', justifyContent: 'center' }}>
  <PaperComponent color="secondary" />
  <PaperComponent color="pink" />
  <PaperComponent color="#655eba" />
  <PaperComponent color="neutral-dark" />
</div>
