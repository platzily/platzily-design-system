# Papers

Paper is a basic component used to define some properties to have consistency in container elements.

<div align="center">
  <PaperComponent size={{ width: 300, height: 100 }} elevation="1" />
  <PaperComponent size={{ width: 300, height: 100 }} color="secondary" />
  <PaperComponent size={{ width: 300, height: 100 }} elevation="4" color="neutral" />
  <PaperComponent size={{ width: 300, height: 100 }} variant="outlined" />
</div>

### Basic Paper

```jsx
import { Paper }from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <PaperComponent size='200px' color='neutral-secondary' />
    <PaperComponent size='200px' color='neutral' />
    <PaperComponent size='200px' color='error' />
    <PaperComponent size='200px' color='info' />
  )
}

```


<div style={{display: "grid", gridTemplateColumns: "auto auto", justifyContent: "center"}}>
<PaperComponent size='200px' color='neutral-secondary'/>
<PaperComponent size='200px' color='neutral'/>
<PaperComponent size='200px' color='error'/>
<PaperComponent size='200px' color='info'/>
</div>

### Variant
You could have an outlined paper.

```jsx
import { Paper }from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <PaperComponent size='200px' variant="outlined" />
  )
}
```

<div align="center">
  <PaperComponent size='200px' variant="outlined" />
</div>

### Elevation

The elevation can be used to establish a hierachy between other content. In practical terms, the elevation controls the size of the shadow applied to the surface.

```jsx
import { Paper }from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <PaperComponent elevation="0">elevation=0</PaperComponent>
    <PaperComponent elevation="1">elevation=1</PaperComponent>
    <PaperComponent elevation="2">elevation=2</PaperComponent>
    <PaperComponent elevation="3">elevation=3</PaperComponent>
    <PaperComponent elevation="4">elevation=4</PaperComponent>
    <PaperComponent elevation="6">elevation=6</PaperComponent>
    <PaperComponent elevation="8">elevation=8</PaperComponent>
    <PaperComponent elevation="12">elevation=12</PaperComponent>
  )
}

```

### Size

With Size prop, you can change the size of Paper to taste. You could compose using the props approach like this:
```jsx
import { Paper }from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <PaperComponent size='200px' color='neutral-secondary'/>
    <PaperComponent size={{width: 400, height: 200}} color='primary'/>
  )
}

```

The code above will result in:

<div style={{display: "grid", gridTemplateColumns: "auto auto", justifyContent: "center"}}>
<PaperComponent size='200px' color='neutral-secondary'/>
<PaperComponent size={{ width: 400, height: 200 }} color='primary'/>
</div>


## Color

The `Paper` change of the color with `color` prop and use values as hexcolor, [colors default](/platzily-ui/colors), HTML color groups, is by default: 'neutral-tertiary'


```jsx
import { Paper }from '@platzily-ui/components';

const MyPaper = () => {
  return (
    <PaperComponent size='200px' color='secondary'/>
    <PaperComponent size='200px' color='pink'/>
    <PaperComponent size='200px' color='#655EBA'/>
    <PaperComponent size='200px' color='neutral-dark'/>
  )
}

```

<PaperComponent size='200px' color='pink'/>
<PaperComponent size='200px' color='#655eba'/>
<PaperComponent size='200px' color='neutral-dark'/>
</div>
